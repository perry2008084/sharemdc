# ShareMDC 更新日志

## [v1.1.0] - 2026-03-23

### 新增功能 ✨

- **一键导出功能**
  - 支持导出为 Markdown 格式（.md 文件）
  - 支持导出为 PDF 格式（通过打印对话框）
  - 编辑页面和分享页面均支持导出
  - 点击"导出"按钮打开导出菜单，选择导出格式

- **多语言支持扩展**
  - 为所有 7 种支持的语言添加了导出功能的翻译
  - 支持的语言：中文、英语、日语、德语、西班牙语、法语、葡萄牙语

### 技术改进 🔧

- 新增 `export.js` 模块，提供导出功能
- 在 `index.html` 和 `share.html` 中添加导出按钮
- 更新 `i18n.js` 添加导出相关翻译
- 更新 `styles.css` 添加导出菜单和通知样式
- 改进 `app.js` 和 `share.js` 集成导出功能

### 用户体验优化 🎨

- 导出菜单带有优雅的动画效果
- 导出成功/失败时显示通知提示
- PDF 导出自动优化打印样式，隐藏复制按钮等非必要元素
- 导出的 Markdown 文件自动命名为 `document.md` 或 `shared-{id}.md`

### 使用说明 📖

#### 在编辑页面导出：
1. 点击右上角的"导出"按钮
2. 选择"下载 Markdown"或"打印为 PDF"
3. 完成导出

#### 在分享页面导出：
1. 点击右上角的"导出"按钮
2. 选择"下载 Markdown"或"打印为 PDF"
3. 完成导出

### 注意事项 ⚠️

- PDF 导出使用浏览器原生打印功能，用户可以在打印对话框中选择"保存为 PDF"
- PDF 导出会自动隐藏代码复制按钮等非打印元素
- 打印样式已优化，支持代码块自动换行
